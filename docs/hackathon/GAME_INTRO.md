# 두 문 러너 (Door Runner) — 게임 소개 및 설명 문서

> NAN 2026 (NHN Game × AI Hackathon) 사전과제 제출물 3
> 제출자: 김학현 (개인 참여)

## 1. 게임 제목 및 한 줄 소개

**두 문 러너 (Door Runner)** — 정답이 적힌 문을 몸으로 열며 끝없이 달리는 3D 초등 수학 러너 게임.

## 2. 게임 개요

- 초등학교 1학년 1학기 ~ 6학년 2학기 **전 학기 수학 교육과정**을 다루는 학습 게임입니다. 학기당 20문제, 총 **240문제**의 자체 제작 문제 은행을 탑재했습니다 (예: 2-2 곱셈구구, 4-1 각도, 6-2 분수÷분수·비례식).
- 캐릭터가 3D 트랙을 끝없이 달리다가 두 개의 문 앞에 도착하면 문제판이 나타나고, 정답이 적힌 문 쪽으로 직접 걸어가 몸으로 문을 열면 통과하는 방식입니다. 문제를 "고르는" 것이 아니라 "몸으로 달려가 여는" 조작이 핵심 재미입니다.
- 오답이어도 탈락시키지 않고 정답을 보여준 뒤 **같은 문제에 재도전**시키는, 학습 강화를 우선한 설계입니다.
- 모바일 우선 웹앱 (React + three.js) — 브라우저만 있으면 어떤 기기에서든 실행됩니다.

## 3. 게임 방법

### 목표
문제판을 읽고 정답이 적힌 문을 골라 통과하며, 최대한 많은 문을 통과해 ⭐(별)과 최고 기록을 쌓는 것.

### 조작
| 상황 | 모바일 | 데스크톱 (키보드) |
|------|--------|--------|
| 문 선택 구간 | 화면 **왼쪽/오른쪽을 꾹** 누르면 그 방향 문으로 걸어감 | `←`/`→` 또는 `A`/`D` |
| 앞으로 걷기 | — | `↑`, `W`, 스페이스 |

달리기 구간은 자동 진행이며, 문 앞 구간에서만 좌우 조작으로 문을 향해 걸어갑니다. 문 사이 벽은 막혀 있어 반드시 한쪽 문을 골라야 합니다.

### 규칙
- **첫 시도에 정답** 문을 통과하면 ⭐ 1개 적립.
- 오답 문에 닿으면 💥 정답을 알려주고 💖(하트) 1개 감소 → 같은 문제 앞에서 재도전 (문 좌우 배치는 재배치).
- 💖는 3개로 시작. 학기당 20문제를 다 풀면 재셔플되어 무한히 이어집니다.

### 종료 조건
💖 0개가 되면 게임오버 — 통과한 문 개수와 최고 기록이 표시되고, "다시 달리기"로 즉시 재시작할 수 있습니다.

## 4. 실행 방법

### 웹 (권장 — 설치 불필요)
아래 링크를 브라우저(모바일/PC)에서 열면 바로 플레이됩니다.

> **플레이 링크: https://hakhyun-kim.github.io/door-runner/**

첫 화면에서 학기(1-1 ~ 6-2)를 고르면 게임이 시작됩니다. 별도 로그인·설치·유료 라이선스가 필요 없습니다.

### 로컬 실행 (소스 코드)
```bash
git clone https://github.com/Hakhyun-Kim/door-runner.git
cd door-runner
npm install
npm run dev   # http://localhost:5174
```
요구 사항: Node.js 20 이상.

## 6. 기술 요약

| 항목 | 내용 |
|------|------|
| 렌더링 | three.js + @react-three/fiber (3D 러너 씬, 절차 생성 지오메트리) |
| 프레임워크 | React 18 + TypeScript + Vite |
| 텍스트 렌더링 | 한글 텍스트 → 캔버스 → 텍스처 (폰트 파일 없이 웹폰트/시스템 폰트) |
| 사운드 | Web Audio API 실시간 합성 (오디오 파일 없음) |
| 데이터 | 백엔드 없음 — localStorage (학기, 누적 별, 최고 기록, 음소거) |
| 배포 | GitHub Actions → GitHub Pages 자동 배포 |
