## 남은 진행 순서

1. **플레이 점검** — 배포된 링크를 데스크톱 + 모바일 브라우저에서 실제 플레이 확인. 심사자는 PC 브라우저로 열 가능성이 높으므로 키보드 조작(←/→·A/D·↑) 안내가 화면에 보이는지, 가로 화면에서도 문제없는지 확인.
2. **게임 폴리싱 업데이트** — 점검에서 나온 어색한 부분 수정 (커밋 기록이 곧 개발 과정 증빙이므로 이 저장소에서 작업).
3. **플레이 영상 촬영·업로드** (아래 가이드)
4. **문서 2종 PDF 변환·최종화** — 영상 링크 확정 후 `GAME_INTRO.md`, `AI_USAGE.md`의 빈칸을 채우고 PDF로 변환.
5. **최종 점검** — 제출 직전 시크릿 창에서 Pages 링크·YouTube 링크·저장소 접근 재확인.

## 플레이 영상 촬영 가이드 (30~60초)

- AI 합성·편집 조작 불가 — **실제 플레이 화면 그대로** 녹화 (모바일 화면 녹화 또는 데스크톱 OBS/Win+G).
- 추천 구성 (총 ~50초):
  1. (0–5초) 학기 선택 화면 — 1학년~6학년 전 학기 커버를 한눈에
  2. (5–30초) 핵심 루프 2~3문제 — 달리기 → 문제 읽기 → 정답 문으로 걸어가 통과 → ⭐·폭죽
  3. (30–40초) 오답 장면 1회 — 💥 정답 표시 → 💖 감소 → 같은 문제 재도전 (학습 강화 컨셉 어필)
  4. (40–50초) 게임오버 → 최고 기록 → 다시 달리기
- 소리 포함 녹화 권장 (효과음이 Web Audio 자체 합성이라는 점이 특징).
- YouTube에 **공개 또는 일부 공개(링크 공유)**로 업로드 → 심사 종료까지 유지.
